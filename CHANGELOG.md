# Changelog

All notable changes to Flowreach are documented here.

Format: `[version] — date` followed by what changed.

---

## [1.0.0] — 2025

Initial public release.

### Features
- Import `.xlsx` exports from Outscraper's Google Maps Scraper
- Prospect table with sortable columns (Company, Reviews, Rating, City)
- Drag-and-drop column reordering
- Filter by pipeline stage
- Call mode via `tel:` links (iPhone Handoff compatible)
- Copy mode with personalised SMS templates
- Placeholders: `(name)`, `(city)`, `(trade)` — auto-filled per lead on copy
- Up to 5 SMS templates with per-lead script tracking
- Up to 5 call scripts with in-panel preview
- Customisable pipeline stages (add, rename, reorder, colour)
- Column visibility toggle (Settings panel)
- Per-lead detail panel with notes and activity timeline
- Status change logging with timestamps
- Global search across all batches (name or phone number)
- Drag-to-reorder batches in sidebar
- Batch rename via ✎ button
- Statistics view: KPI cards, status breakdown, conversion funnel, script comparison
- PDF batch report generation
- Excel export (visible columns only, with custom stage labels)
- Logo upload (stored locally)
- All data stored in browser `localStorage` — no servers, no accounts
- MIT licensed, single HTML file, zero build process
