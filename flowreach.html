Flowreach · Getting started

The outreach tool built for
Outscraper lists.

Import a Google Maps lead list from Outscraper's Google Maps Scraper, and Flowreach turns it into a structured calling and texting workspace — tracked, organised, and ready to go.

💾 Data stays in your browser 🔒 No accounts · No servers · Free
How it works
📥
Import

Buy a list from Outscraper's Google Maps Scraper and drop the .xlsx file here.

📞
Call & text

One click to call via iPhone Handoff, or copy a personalised SMS and send from your phone.

📊
Track

Update status per lead. Statistics shows reply rates, conversion and script performance.

Calling via iPhone Handoff

Switch to Call mode in the Prospects toolbar. Every number becomes a tap-to-call link. Your Mac triggers the call, your iPhone handles it.

Requirements
✓Same iCloud account
✓Same Wi-Fi network
✓Bluetooth on both
✓Handoff enabled
Settings → General → AirPlay & Handoff → on

Getting started — 5 steps
1
Buy a list from Outscraper

Go to outscraper.com → Google Maps Scraper. Search your niche and city (e.g. "plumbers Stockholm"), export as .xlsx.

2
Import into Flowreach

Click Import in the sidebar. Drop the file, name the batch, click Import. Rows without phone numbers are filtered automatically.

3
Write your SMS templates

Go to SMS Templates. Use (name) (city) (trade) — filled automatically on copy.

4
Work the list

Open Prospects. Call or text each lead. Click any row for full details, notes, and activity timeline. Update status as you go.

5
Review results

Go to Statistics for reply rates, meeting rates, and script comparison. Export to Excel or PDF from the Prospects toolbar.

Tips
→

Batches — drag to reorder in the sidebar. Click ✎ to rename. Each batch has its own pipeline and statistics.

→

Search — the sidebar search finds any lead by name or number across all batches instantly.

→

Pipeline stages — customise in Settings. Add stages, rename them, assign colours. Filter chips update automatically.

→

Data safety — everything lives in your browser. Bookmark this page and export to Excel regularly as a backup.

→

Columns — drag headers to reorder. Hide columns you don't use in Settings → Column visibility.

Is Flowreach safe to use?

Yes. Flowreach is a single HTML file that runs entirely in your browser. No servers, no cloud storage, no analytics, nothing leaves your device. Source code is fully open on GitHub for anyone to verify.
